# UI & Design System Protocol

1. **Source of Truth:** Το αρχείο `[docs/design-system.md]` είναι η απόλυτη πηγή αλήθειας για το UI/UX, τα components (Shadcn), το styling (Tailwind) και το typography.
2. **Read Before Act:** Πριν γράψεις ή τροποποιήσεις οποιοδήποτε React/Next.js component, ΠΡΕΠΕΙ να διαβάζεις αυτό το αρχείο για να διασφαλίσεις την οπτική και δομική συνέπεια.
3. **Auto-Update Mechanism:** Εάν κατά τη διάρκεια της συνομιλίας μας (vibe coding) αποφασίσουμε να αλλάξουμε ένα design pattern, να προσθέσουμε ένα νέο χρώμα, ή να φτιάξουμε ένα νέο επαναχρησιμοποιήσιμο component, η **τελευταία σου ενέργεια** πρέπει πάντα να είναι η αυτόματη ενημέρωση του `[path/to/your/design-system.md]` ώστε να αντικατοπτρίζει τη νέα απόφαση.
